# SEGA MEGADRIVE AV CABLE

## CONNECTOR (Cable side)

```
  _______
 / 7   1 \
| 6  8  2 |
|  5   3  |
 \   4   /
  -------
```

## PIN ASSGIN

|No|Detail|
|:-:|:-:|
|1|Composit/RGB(?) Sync|
|2|Composit Video|
|3|RGB Green|
|4|GND|
|5|5V|
|6|Audio Mono|
|7|RGB Red|
|8|RGB Blue|

Note that DIN8 has U-type(262 degrees) and C-type(270 degrees). MEGADRIVE is U-type.
