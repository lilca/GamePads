# SEGA MEGADRIVE AV CABLE

## CONNECTOR (Cable side)

```
  _______
 / 7   8 \
| 1  6  3 |
|  4   5  |
 \   2   /
  -------
```

## PIN ASSGIN

|No|Detail|
|:-:|:-:|
|1|Audio Mono|
|2|GND|
|3|Composit Video|
|4|5V|
|5|RGB Green|
|6|RGB Blue|
|7|RGB Red|
|8|Composit/RGB(?) Sync|

Note that DIN8 has U-type(262 degrees) and C-type(270 degrees). MEGADRIVE is U-type.
