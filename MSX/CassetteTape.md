# ボーレート
|ボーレート[bps]|特徴|
|:-:|:--|
|1200|低スピード・高信頼|
|2400|高スピード・低信頼|

# １ビット
|ボーレット[bps]|ビット|波形|説明|
|:-:|:-:|:-|:-:|
|1200|0|\_\_\_\_----|1200Hz x 1wave|
|1200|1|\_\_--\_\_--|2400Hz x 2wave|
|2400|0|\_\_--|2400Hz x 1wave|
|2400|1|\_-\_-|4800Hz x 2wave|
